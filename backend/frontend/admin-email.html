邮箱与 SMTP 配置

用户提交联系表单后，通知邮件将发送到此邮箱

SMTP 服务器配置

配置 SMTP 服务器以发送邮件通知

常见：smtp.163.com, smtp.qq.com, smtp.gmail.com

常用端口：465 (SSL) 或 587 (TLS)

通常是完整的邮箱地址

163/QQ邮箱需使用授权码，而非登录密码

邮件中显示的发件人地址，通常与 SMTP 用户名相同

📮 常见邮箱 SMTP 配置

163 邮箱
  • • 服务器：smtp.163.com
  • • 端口：465 (SSL) 或 25
  • • 需要开启 SMTP 服务并使用授权码
QQ 邮箱
  • • 服务器：smtp.qq.com
  • • 端口：465 (SSL) 或 587 (TLS)
  • • 需要开启 SMTP 服务并使用授权码
Gmail
  • • 服务器：smtp.gmail.com
  • • 端口：465 (SSL) 或 587 (TLS)
  • • 需要开启"允许不够安全的应用"或使用应用专用密码

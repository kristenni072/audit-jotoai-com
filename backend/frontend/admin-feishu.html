/ 飞书集成配置

飞书机器人 Webhook

用户提交联系表单时，将发送通知到飞书群聊

飞书多维表格同步

打开多维表格，复制浏览器地址栏中的完整 URL

📘 配置步骤

1. 创建飞书自定义机器人（可选）
  • • 在飞书群聊中，点击"设置" → "群机器人" → "添加机器人"
  • • 选择"自定义机器人"，设置名称和头像
  • • 复制生成的 Webhook 地址
2. 创建飞书应用
  • • 访问 飞书开放平台
  • • 创建企业自建应用
  • • 在"凭证与基础信息"中获取 App ID 和 App Secret
3. 配置应用权限
  • • 进入应用管理 → "权限管理"
  • • 添加权限：bitable:app（查看、评论、编辑和管理多维表格）
  • • 添加权限：base:record:create（新增记录）
  • • 创建版本并发布应用
4. 将应用添加到多维表格
  • • 打开目标多维表格
  • • 点击右上角"..." → "更多" → "添加文档应用"
  • • 搜索并添加您创建的应用
  • • 授予应用"可编辑"权限
5. 表格字段要求
  • • 网址（文本）
  • • 日期（文本）
  • • 姓名（文本）
  • • 公司/学校/机构（文本）
  • • 手机号码（文本）
  • • 电子邮箱（文本）
  • • 咨询需求（文本）
  • • 来源（文本）
